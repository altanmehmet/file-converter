# Icon Dosyaları

Bu klasörde aşağıdaki icon dosyaları olmalı:

## Zorunlu Dosyalar

1. **favicon.ico** - Browser tab icon (16x16, 32x32, 48x48)
2. **icon-192.png** - PWA icon (192x192)
3. **icon-512.png** - PWA icon (512x512)
4. **apple-touch-icon.png** - iOS icon (180x180)
5. **og-image.png** - Open Graph image (1200x630)

## Oluşturma

Detaylı bilgi için `icon-generator.md` dosyasına bak.

## Hızlı Başlangıç

1. https://favicon.io/favicon-generator/ adresine git
2. "Quick Convert" yaz veya logo yükle
3. Tüm dosyaları indir
4. Bu klasöre kopyala

OG Image için:
1. Canva veya benzeri araç kullan
2. 1200x630px boyutunda tasarla
3. "og-image.png" olarak kaydet
4. Bu klasöre kopyala
