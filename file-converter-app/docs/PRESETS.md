# PRESETS.md

## Documents
- `doc_to_pdf`: `.doc, .docx -> .pdf`
- `pdf_to_docx` (Beta): `.pdf -> .docx`
- `excel_to_pdf`: `.xls, .xlsx -> .pdf`
- `ppt_to_pdf`: `.ppt, .pptx -> .pdf`

## PDF
- `pdf_compress`: `.pdf -> .pdf` (`low|medium|high`)
- `pdf_merge`: birden fazla `.pdf -> merged .pdf`
- `pdf_split`: `.pdf -> birden fazla .pdf` (ranges/every)

## Images
- `image_convert`: `.png/.jpg/.jpeg/.webp/.tiff -> webp/png/jpg`
- Opsiyonlar: `quality`, `maxWidth`, `stripMetadata`

## Notlar
- `pdf_to_docx` kalite acisindan kaynak PDF'e bagimlidir.
- `pdf_merge` multi-file zorunlu preset'tir.
